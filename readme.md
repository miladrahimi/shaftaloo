# Shaftaloo!
Shaftaloo is a simple cost accounting web (backend + PWA) application.

## Installation

```bash
$ git clone https://github.com/miladrahimi/shaftaloo
$ cd shaftaloo
$ cp .env.example .env
$ docker-compose up -d
```
